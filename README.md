
This is a simple IRC bot, made with love by and for the [SLE](http://sle.ec)

Installation
-----------

Library depends

* python-twisted
* python-yapsy
* python-tweepy

    bash devel_setup.sh

Usage
-----

### Activate the virtual enviroment (optional)

    source venv/bin/activate

### Edit the bot.cf file to your convenience


### Run bot.py

    python bot.py bot.cf
